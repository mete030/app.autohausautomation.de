# app.autohausautomation.de

Wichtiger Hinweis: Diese App wird lokal immer unter `http://localhost:3007/` betrieben.

- Verwende lokal immer `http://localhost:3007/`
- Andere lokale Ports sind in dieser Umgebung belegt

## Lokaler Start

```bash
npm install
npm run dev
```

Danach ist die App unter `http://localhost:3007/` erreichbar.

## Production Build lokal testen

```bash
npm run build
npm run start
```

Auch der Start-Server läuft lokal auf `http://localhost:3007/`.
